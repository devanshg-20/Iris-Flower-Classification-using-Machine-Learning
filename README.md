# Iris Flower Classification using Machine Learning 

## Project Overview
This project implements three machine learning algorithms to classify Iris flowers into their species based on petal and sepal measurements.  
The algorithms used are:

- Logistic Regression  
- K-Nearest Neighbors (KNN)  
- Gaussian Naive Bayes  

The goal is to train models on the Iris dataset and compare their performance using **cross-validation** and **test accuracy**.

---

## Dataset
- Dataset used: [Iris Dataset] 
- Included in the repository as `Iris.csv`  
- Features:
  - Sepal Length (cm)  
  - Sepal Width (cm)  
  - Petal Length (cm)  
  - Petal Width (cm)  
- Target:
  - Species (Iris-setosa, Iris-versicolor, Iris-virginica)
